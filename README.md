# Roomlo

A full-stack project with separate `client` and `server` directories.

## 📦 Prerequisites

- Node.js (v14+ recommended)
- npm

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name


### 2. Start Client
```bash
cd server
npm install
npm run dev


### 3. Start Server
```bash
cd client
npm install
npm run dev
